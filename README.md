# Resume Parsing & Screening Results
The file resume_parsing_screening_output.csv contains the final output generated after processing resumes through the AI-based screening system.

This dataset represents structured candidate information extracted from resumes along with their evaluation scores based on job requirements.

Contents of the File

The CSV file includes the following key fields:

Candidate Name
Extracted Skills
Education Details
Work Experience
Keywords Matched
Screening Score / Ranking
Selection Status (Shortlisted / Rejected)
Purpose

This output file helps recruiters:

Quickly compare candidates
Identify top-performing profiles
Make data-driven hiring decisions
Reduce manual resume screening effort
How It Was Generated
Resumes were uploaded and processed in Google Colab
NLP techniques extracted relevant information
Candidate data was matched with job requirements
A scoring mechanism ranked candidates
Final structured results were exported as a CSV file
File Format
Format: CSV (Comma Separated Values)
Easy to open in Excel, Google Sheets, or Power BI
Ready for further analysis or visualization
